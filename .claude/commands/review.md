# Review Command

Route changed files to relevant review agents.

## File Routing
- `lib/src/models/` → architecture-review, code-review
- `lib/src/controller/` → architecture-review, code-review
- `lib/src/widgets/` → architecture-review, code-review
- `test/` → testing-review
- `*.dart` → code-review (always)
- JSON parsing changes → security-review
